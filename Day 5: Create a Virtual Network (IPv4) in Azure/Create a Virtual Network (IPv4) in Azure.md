Create a Virtual Network (VNet) named `xfusion-vnet` in the `East us region` with `192.168.0.0/24` IPv4 CIDR.

Steps:

1. Sign in to the [Azure Portal](https://portal.azure.com/) using the provided credentials

2. Use the global search bar to search for “Virtual network.” From the results, select “Virtual network.”

<img width="993" height="879" alt="1" src="https://github.com/user-attachments/assets/42c4d35f-59e5-42b6-aae1-a36298715809" />

4. From the Virtual Network dashboard, click on create button

<img width="957" height="875" alt="2" src="https://github.com/user-attachments/assets/ae06d70b-3cd2-4400-9dc7-39d15429cd81" />

5. On the Basics tab of the Create virtual network page, enter the VM name and select the appropriate region.

<img width="961" height="874" alt="3" src="https://github.com/user-attachments/assets/7901c883-1344-4fb4-af48-4782dfd9ef2f" />

6. On the IP Addresses tab of the Create virtual network page, enter the given IP

<img width="961" height="861" alt="4" src="https://github.com/user-attachments/assets/9342af70-1d35-4873-bf98-a561be7110c9" />

7. Then, click on create button

<img width="955" height="869" alt="5" src="https://github.com/user-attachments/assets/224fe253-df08-4fe3-9c4d-2d8299901c48" />
