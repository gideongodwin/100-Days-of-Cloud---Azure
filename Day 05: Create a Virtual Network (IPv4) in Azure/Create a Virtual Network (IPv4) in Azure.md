## Day 5: Create a Virtual Network (IPv4) in Azure

#### Task Details:

Create a Virtual Network (VNet) named `xfusion-vnet` in the `East us region` with `192.168.0.0/24` IPv4 CIDR.

#### STEPS:

1. Sign in to the [Azure Portal](https://portal.azure.com/) 

2. Use the global search bar to search for “Virtual network.” From the results, select “Virtual network.”

<img width="1151" height="425" alt="537214353-e8979721-e595-445d-b990-ff309837bbcc" src="https://github.com/user-attachments/assets/a0f65fce-b49e-4578-8eef-33963b352881" />

3. From the Virtual Network dashboard, click on create button

<img width="957" height="875" alt="2" src="https://github.com/user-attachments/assets/ae06d70b-3cd2-4400-9dc7-39d15429cd81" />

4. On the Basics tab of the Create virtual network page, enter the VM name and select the appropriate region.

<img width="961" height="874" alt="3" src="https://github.com/user-attachments/assets/7901c883-1344-4fb4-af48-4782dfd9ef2f" />

5. On the IP Addresses tab of the Create virtual network page, enter the given IP

<img width="961" height="861" alt="4" src="https://github.com/user-attachments/assets/9342af70-1d35-4873-bf98-a561be7110c9" />

6. Then, click on create button
